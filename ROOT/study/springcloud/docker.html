# Docker安装

# 一: 安装

注意

在新主机首次安装 Docker Engine-Community之前，需要设置Docker仓库，之后，您可以从仓库安装和更新 Docker。

设置仓库，需要安装所需的软件包。yum-utils 提供了 yum-config-manager ，并且 device mapper 存储驱动程序需要 device-mapper-persistent-data 和 lvm2。

  1. 安装依赖包
 sudo yum install -y yum-utils device-mapper-persistent-data lvm2
1
  1. 设置阿里云镜像
sudo yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
1
  1. 安装Docker-CE
sudo yum install docker-ce
1
  1. 启动Docker-CE和设置自启动
sudo systemctl start docker
1
sudo systemctl enable docker
1

# 二: 设置

  1. Docker镜像加速

  • 网易: https://hub-mirror.c.163.com/

  • 阿里云: https://<你的ID>.mirror.aliyuncs.com

  • 七牛云加速器: https://reg-mirror.qiniu.com

  • Docker中国区官方镜像: https://registry.docker-cn.com

  • 中国科技大学: https://docker.mirrors.ustc.edu.cn

在 /etc/docker/daemon.json 文件（没有就自己去创建一下这个文件！）内填写如下内容:

{
  "registry-mirrors":["https://ung2thfc.mirror.aliyuncs.com"]
}
1
2
3
  1. 然后加载和重启Docker
sudo systemctl daemon-reload
1
sudo systemctl restart docker
1
  1. 查看是否成功(Mirror)
docker info

# 三: 常用命令

  1. 拉取镜像
docker pull <镜像名:tag>
# 例如以下代码
docker pull jenkins/jenkins:ls
1
2
3
  1. 删除容器
docker rm <容器名 or ID>
1
  1. 查看容器日志
docker logs -f <容器名 or ID>
1
  1. 查看正在运行的容器
docker ps
docker ps -a # 为查看所有的容器，包括已经停止的。
1
2
  1. 删除所有容器
docker rm $(docker ps -a -q)
1
  1. 停止、启动、杀死指定容器
docker start <容器名 or ID> # 启动容器
docker stop <容器名 or ID> # 启动容器
docker kill <容器名 or ID> # 杀死容器
1
2
3
  1. 查看所有镜像
docker images
1
  1. 后台运行
docker run -d <Other Parameters>
# 例如
docker run -d -p 127.0.0.1:33301:22 centos6-ssh
1
2
3
  1. 暴露端口
# 一共有三种形式进行端口映射
docker -p ip:hostPort:containerPort # 映射指定地址的主机端口到容器端口
# 例如：docker -p 127.0.0.1:3306:3306 映射本机3306端口到容器的3306端口
docker -p ip::containerPort # 映射指定地址的任意可用端口到容器端口
# 例如：docker -p 127.0.0.1::3306 映射本机的随机可用端口到容器3306端口
docer -p hostPort:containerPort # 映射本机的指定端口到容器的指定端口
# 例如：docker -p 3306:3306 # 映射本机的3306端口到容器的3306端口
