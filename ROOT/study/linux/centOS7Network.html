# centOS7 mini版设置网络

# 方式一 类似图形化修改

首先用命令查看当前的网卡状态

nmcli d
1

然后使用命令，并入下图所示方式修改

nmtui
1
点击查看操作图片

  • 上一步操作完，使用service network restart重启一下网络即可使用nmcli d查看状态。

# 方式二 修改配置文件方式

  1. 同上，查看状态。
  2. 使用命令cd /etc/sysconfig/network-scripts进入到指定文件夹。
  3. 用ls 查看存在的文件，使用vi ifcfg-ens33来修改文件（注意，不一定是叫这个名字，根据实际情况定！），进入后使用i开启编辑，修改完成后，按ESC，:wq退出，然后使用service network restart重启一下网络即可使用nmcli d查看状态。
