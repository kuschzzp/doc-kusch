    • Navicat连接(记得关闭Linux上的防火墙)

    • 创建用户(用户名@连接ip by 密码)

        `create user 'springuser'@'localhost' identified by 'password'`

    • 修改用户可访问的host域 use mysql;
      update user set host='%(或者指定IP)' where user = 'root';

    • 授权 grant all privileges on *.* to root@'%（或指定IP）' identified by 'password' with grant option;
      *.* 代表 数据库.表 收回权限
      revoke all on *.* from root@'%';
      查看某个用户的权限
      show grants for 'nextcloud'@'%';
      or
      select * from mysql.user where user='root';

    • 授予某个文件可执行权限 chmod -x 文件

    • 访问问题 虚拟机用：curl 192.168.199.200:8080 可以访问 但是主机访问不了

        	设置将数据转发到本机的其他网卡设备上就可以了
  	iptables -P FORWARD ACCEPT

  	同时再检查一下防火墙关了没，没关就关一下
  	systemctl status firewalld.service
