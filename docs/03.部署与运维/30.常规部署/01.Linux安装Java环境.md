## Linux安装Java环境

## 命令一键安装 jdk8，并配置好环境变量

```shell
cd /opt/ && yum install -y wget && wget http://cdn.superkusch.fun/docs/jdk-8u341-linux-`uname -p`.tar.gz -O jdk8u341.tar.gz && tar -zxvf jdk8u341.tar.gz && rm -f jdk8u341.tar.gz && echo "export JAVA_HOME=`pwd`/jdk1.8.0_341" >> /etc/profile && echo 'export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar' >> /etc/profile && echo 'export PATH=${JAVA_HOME}/bin:$PATH' >> /etc/profile && source /etc/profile && java -version
```
