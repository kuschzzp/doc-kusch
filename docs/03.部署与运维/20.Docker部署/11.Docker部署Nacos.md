### 单机部署
- 创建一些文件先 (可以先创建application.properties然后复制粘贴官网的内容进去，也可以下载好放进去。)
```shell
mkdir -p /opt/nacos/conf && \
touch /opt/nacos/conf/application.properties 
```

- 配置文件下载处：[application.properties](https://github.com/alibaba/nacos/blob/3.1.1/bootstrap/src/main/resources/application.properties)

- 命令

> 一个不错的docker镜像站 [https://1ms.run/](https://1ms.run/)

```shell
docker run -d \
-v /etc/localtime:/etc/localtime:ro \
-v /opt/nacos/conf/application.properties:/home/nacos/conf/application.properties \
-e PREFER_HOST_MODE=ip \
-e MYSQL_HOST=数据库host \
-e NACOS_DATABASE=k_nacos \
-e MYSQL_PORT=3306 \
-e MYSQL_USER=root \
-e MYSQL_PASSWORD=数据库密码 \
-e NACOS_AUTH_TOKEN=OWEzYmRjNjAzYzk0MGExODQ2N2QxNjdhZjE1ZDRjOGM= \
-e NACOS_AUTH_IDENTITY_KEY=NACOSAUTHIDENTITYKEY \
-e NACOS_AUTH_IDENTITY_VALUE=543D36086967BAE7AC7BE0E8DC277E8B543D36086967BAE7AC7BE0E8DC277E8B \
-p 18848:8848 -p 19848:9848 -p 18080:8080 \
--name nacos311 \
--privileged=true \
--restart=always \
nacos/nacos-server:v3.1.1
```

- NACOS_AUTH_TOKEN -- > nacos.core.auth.plugin.nacos.token.secret.key
- NACOS_AUTH_IDENTITY_KEY -- > nacos.core.auth.server.identity.key
- NACOS_AUTH_IDENTITY_VALUE -- > nacos.core.auth.server.identity.value

> 初次访问 **http://xxxx:18080/index.html** ,然后设置默认的账号和密码。
