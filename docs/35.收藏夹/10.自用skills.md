## 自用的简略型技能配置。

下面是安装在全局的几个技能，相较于直接安装全量 superpowers 会节省一些token。

> npx skill 使用  [https://www.npmjs.com/package/skills](https://www.npmjs.com/package/skills)  
> 下面的都是安装在codex上。

- 头脑风暴，编程开始前提出想法验证想法，需求澄清。
```shell
npx skills add https://github.com/obra/superpowers -g --skill brainstorming -a codex
```

- 规划 计划、任务待办，按照文件维度设计修改思路。

**这里还可以去把superpowers里面的有关subagent的几个skill安装一下，提速编码，但我觉得没必要**

```shell
npx skills add https://github.com/obra/superpowers -g --skill writing-plans -a codex
```

- 让AI做架构设计，复杂项目领域建模 （实现干净的架构、六边形架构和领域驱动的设计）
```shell
npx skills add https://github.com/wshobson/agents -g --skill architecture-patterns -a codex
```

- 微服务架构设计
```shell
npx skills add https://github.com/wshobson/agents -g --skill microservices-patterns -a codex
```

- 前端项目设计优化，不是专门做“页面设计”的工具，但会影响页面结构和实现方式
```shell
npx skills add https://github.com/vercel-labs/agent-skills -g --skill vercel-react-best-practices -a codex
```

- 前端编程可扩展编码设计模式
```shell
npx skills add https://github.com/wshobson/agents -g --skill design-system-patterns -a codex
```

- 页面视觉UI设计
```shell
npx skills add https://github.com/nextlevelbuilder/ui-ux-pro-max-skill -g --skill ui-ux-pro-max -a codex
```

- 页面排版设计
```shell
npx skills add https://github.com/wshobson/agents -g --skill visual-design-foundations -a codex
```

- 让AI写的文档更好看，像纸质报纸一样。开源项目：[https://github.com/tw93/Kami](https://github.com/tw93/Kami)
```shell
npx skills add tw93/kami -a codex -g -y
```

> 删除：npx skills remove --global xxxx名字xxxx
> 更新：npx skills update -y
