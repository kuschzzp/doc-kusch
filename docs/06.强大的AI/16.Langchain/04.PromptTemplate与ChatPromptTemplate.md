# PromptTemplate 与 ChatPromptTemplate

PromptTemplate 用来构建可复用提示词，避免手写字符串拼接。

## PromptTemplate

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template("请为 {product} 写一句 {style} 风格的广告语。")
value = prompt.invoke({"product": "智能电饭煲", "style": "温暖"})
print(value.text)
```

## ChatPromptTemplate

聊天模型更推荐 `ChatPromptTemplate`。

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个中文广告文案专家。"),
    ("human", "请为 {product} 写一句 {style} 风格的广告语。"),
])

model = ChatOpenAI(model="gpt-4o-mini", temperature=0.7)
chain = prompt | model

response = chain.invoke({"product": "智能电饭煲", "style": "温暖"})
print(response.content)
```

## MessagesPlaceholder

用于插入历史消息。

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个中文助手。"),
    MessagesPlaceholder("history"),
    ("human", "{question}"),
])

value = prompt.invoke({
    "history": [
        {"role": "user", "content": "我叫李雷"},
        {"role": "assistant", "content": "好的，我记住了。"},
    ],
    "question": "我叫什么？",
})

print(value.to_messages())
```

## 总结

- 普通文本模板：`PromptTemplate`
- 聊天模型模板：`ChatPromptTemplate`
- 历史消息占位：`MessagesPlaceholder`
