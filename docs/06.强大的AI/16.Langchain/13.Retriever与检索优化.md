# Retriever 与检索优化

Retriever 是 RAG 中负责“找资料”的组件。

## 基础 Retriever

```python
from langchain_core.documents import Document
from langchain_core.vectorstores import InMemoryVectorStore
from langchain_openai import OpenAIEmbeddings

embeddings = OpenAIEmbeddings(model="text-embedding-3-small")

docs = [
    Document(page_content="Python 是一种编程语言。", metadata={"type": "programming"}),
    Document(page_content="LangChain 用于构建 LLM 应用。", metadata={"type": "ai"}),
    Document(page_content="RAG 通过检索增强模型回答。", metadata={"type": "ai"}),
]

vectorstore = InMemoryVectorStore.from_documents(docs, embeddings)
retriever = vectorstore.as_retriever(search_kwargs={"k": 2})

results = retriever.invoke("什么是 RAG？")
for doc in results:
    print(doc.page_content)
```

## 相似度搜索

```python
results = vectorstore.similarity_search("LLM 应用框架", k=2)
for doc in results:
    print(doc.page_content)
```

## 带分数搜索

```python
results = vectorstore.similarity_search_with_score("LLM 应用框架", k=2)
for doc, score in results:
    print(score, doc.page_content)
```

## MMR 检索

MMR 会平衡相关性和多样性。

```python
retriever = vectorstore.as_retriever(
    search_type="mmr",
    search_kwargs={"k": 2, "fetch_k": 3, "lambda_mult": 0.5},
)

results = retriever.invoke("AI 应用")
for doc in results:
    print(doc.page_content)
```

## 元数据过滤

具体 filter 语法取决于 VectorStore。内存向量库支持简单过滤函数。

```python
def only_ai(doc: Document) -> bool:
    return doc.metadata.get("type") == "ai"

results = vectorstore.similarity_search("应用", k=2, filter=only_ai)
for doc in results:
    print(doc.page_content, doc.metadata)
```

## 总结

检索优化常见手段：

- 调整 `k`
- 使用 MMR
- 使用 metadata filter
- 增加 rerank
- 改写 query
- 优化 chunk 切分
