# 环境安装与包结构

LangChain 1.x 以后，包结构更加模块化。

## Python 版本

建议使用 Python 3.10+。

```bash
python --version
```

## 核心包

```bash
pip install -U langchain langchain-core langsmith
```

常用模型提供商需要单独安装：

```bash
pip install -U langchain-openai
pip install -U langchain-anthropic
pip install -U langchain-google-genai
```

RAG 常用包：

```bash
pip install -U langchain-text-splitters langchain-community
pip install -U langchain-chroma faiss-cpu
```

## 常见包职责

| 包 | 作用 |
| --- | --- |
| `langchain` | Agent、chains、主入口 |
| `langchain-core` | Message、Runnable、Tool、Prompt 等核心抽象 |
| `langchain-openai` | OpenAI 模型与 Embeddings |
| `langchain-community` | 社区集成，版本稳定性弱于专用包 |
| `langchain-text-splitters` | 文本切分器 |
| `langchain-chroma` | Chroma 向量库集成 |
| `langsmith` | Tracing、评估、数据集 |

## 环境变量

```bash
export OPENAI_API_KEY="你的 OpenAI Key"
export LANGSMITH_TRACING=true
export LANGSMITH_API_KEY="你的 LangSmith Key"
export LANGSMITH_PROJECT="langchain-demo"
```

## 验证安装

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-4o-mini", temperature=0)
response = model.invoke("返回 ok")
print(response.content)
```

如果能正常输出，说明基础环境可用。
