# LangSmith 调试与评估

> 官方文档：https://docs.smith.langchain.com/

LangSmith 是 LLM 应用的可观测、调试和评估平台。

它可以记录：

- 模型输入输出
- Prompt
- Tool 调用
- Retriever 返回文档
- Agent 中间步骤
- token 用量
- latency
- 错误堆栈

## 配置环境变量

```bash
export LANGSMITH_TRACING=true
export LANGSMITH_API_KEY="你的 LangSmith Key"
export LANGSMITH_PROJECT="langchain-demo"
```

## 自动追踪 Runnable

```python
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_template("请把下面内容翻译成英文：{text}")
model = ChatOpenAI(model="gpt-4o-mini", temperature=0)
chain = prompt | model | StrOutputParser()

result = chain.invoke({"text": "今天天气真好"})
print(result)
```

配置好环境变量后，这次调用会出现在 LangSmith 项目里。

## 使用 @traceable

非 LangChain 代码也可以追踪。

```python
from langsmith import traceable
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-4o-mini", temperature=0)

@traceable
def summarize(text: str) -> str:
    response = model.invoke(f"请总结这段话：{text}")
    return response.content

print(summarize("LangSmith 可以帮助开发者观察、调试和评估 LLM 应用。"))
```

## 为什么重要？

没有 trace 时，Agent 出错只能看到最终答案。

有 LangSmith 后，可以看到：

```text
用户输入 -> Prompt -> 模型判断 -> Tool 参数 -> Tool 结果 -> 最终回答
```

这对 RAG 和 Agent 特别重要。

## 总结

开发 LLM 应用时，建议从第一天就接入 LangSmith。
