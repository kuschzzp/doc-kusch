# LangServe 与部署

> 官方项目：https://github.com/langchain-ai/langserve

LangServe 可以把 LangChain Runnable 暴露为 HTTP API。

## 安装

```bash
pip install -U "langserve[server]" fastapi uvicorn langchain-openai
export OPENAI_API_KEY="你的 key"
```

## 可运行服务

保存为 `app.py`：

```python
from fastapi import FastAPI
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from langserve import add_routes

prompt = ChatPromptTemplate.from_template("请把下面内容翻译成英文：{text}")
model = ChatOpenAI(model="gpt-4o-mini", temperature=0)
chain = prompt | model | StrOutputParser()

app = FastAPI(title="LangServe Demo")

add_routes(app, chain, path="/translate")

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="127.0.0.1", port=8000)
```

启动：

```bash
python app.py
```

调用：

```bash
curl -X POST http://127.0.0.1:8000/translate/invoke \
  -H "Content-Type: application/json" \
  -d '{"input": {"text": "今天天气真好。"}}'
```

## 常见接口

LangServe 通常会提供：

- `/invoke`
- `/batch`
- `/stream`
- `/input_schema`
- `/output_schema`

## 适合场景

适合：

- Demo
- 内部 API
- 简单 Chain/RAG 服务
- 快速和前端集成

复杂生产 Agent 部署，应评估 LangGraph Platform、LangSmith Deployment 或自定义服务层。

## 总结

LangServe 的定位是：

> 快速把 Runnable 变成 HTTP API。
