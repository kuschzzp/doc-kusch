# ChatModel 与 Messages

ChatModel 是 LangChain 调用聊天模型的统一接口。

## 安装

```bash
pip install -U langchain langchain-openai
export OPENAI_API_KEY="你的 key"
```

## 字符串输入

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-4o-mini", temperature=0)
response = model.invoke("你好，用一句话介绍你自己。")
print(response.content)
```

`invoke()` 返回的是 `AIMessage`，不是普通字符串。

## Messages 输入

真实项目更推荐使用 messages：

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-4o-mini", temperature=0)

messages = [
    {"role": "system", "content": "你是一个简洁的中文技术助手。"},
    {"role": "user", "content": "什么是 ChatModel？"},
]

response = model.invoke(messages)
print(response.content)
```

常见 role：

| role | 含义 |
| --- | --- |
| `system` | 系统指令 |
| `user` | 用户输入 |
| `assistant` | 模型回复 |
| `tool` | 工具返回结果 |

## 常用参数

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(
    model="gpt-4o-mini",
    temperature=0,
    timeout=30,
    max_retries=2,
)

print(model.invoke("返回一句中文问候").content)
```

## 总结

ChatModel 的核心是统一模型调用方式：

```text
messages -> model.invoke() -> AIMessage
```
