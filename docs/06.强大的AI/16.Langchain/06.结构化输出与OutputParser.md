# 结构化输出与 OutputParser

LLM 默认输出文本，但真实业务经常需要 JSON、对象、字段校验。

## StrOutputParser

把 `AIMessage` 转成普通字符串。

```python
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_template("请用一句话介绍 {topic}")
model = ChatOpenAI(model="gpt-4o-mini", temperature=0)
parser = StrOutputParser()

chain = prompt | model | parser

result = chain.invoke({"topic": "LangChain"})
print(type(result))
print(result)
```

## with_structured_output

如果需要结构化对象，推荐使用模型的 `with_structured_output()`。

```python
from pydantic import BaseModel, Field
from langchain_openai import ChatOpenAI

class Person(BaseModel):
    name: str = Field(description="姓名")
    job: str = Field(description="职业")
    city: str = Field(description="所在城市")

model = ChatOpenAI(model="gpt-4o-mini", temperature=0)
structured_model = model.with_structured_output(Person)

result = structured_model.invoke("张三是一名住在杭州的前端工程师。")
print(result)
print(result.name)
```

## Agent 的 response_format

`create_agent` 也支持结构化最终输出。

```python
from pydantic import BaseModel, Field
from langchain.agents import create_agent

class Summary(BaseModel):
    title: str = Field(description="标题")
    points: list[str] = Field(description="要点列表")

agent = create_agent(
    model="openai:gpt-4o-mini",
    tools=[],
    system_prompt="你是一个中文总结助手。",
    response_format=Summary,
)

result = agent.invoke({
    "messages": [{"role": "user", "content": "LangChain 用于构建 LLM 应用，支持模型、工具、RAG 和 Agent。"}]
})

print(result["structured_response"])
```

## 总结

- 想拿字符串：`StrOutputParser`
- 想拿结构化对象：`with_structured_output()`
- Agent 最终结构化输出：`response_format`
