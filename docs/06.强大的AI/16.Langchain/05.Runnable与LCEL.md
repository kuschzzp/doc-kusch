# Runnable 与 LCEL

Runnable 是 LangChain 的统一执行接口。

常见方法：

- `invoke()`：单次调用
- `batch()`：批量调用
- `stream()`：流式调用
- `ainvoke()`：异步调用

## 基础链

```python
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_template("请把这句话翻译成英文：{text}")
model = ChatOpenAI(model="gpt-4o-mini", temperature=0)
parser = StrOutputParser()

chain = prompt | model | parser

result = chain.invoke({"text": "今天天气真好"})
print(result)
```

这里的 `|` 就是 LCEL 管道写法。

## 批量调用

```python
results = chain.batch([
    {"text": "你好"},
    {"text": "谢谢"},
    {"text": "再见"},
])

for item in results:
    print(item)
```

## 流式输出

```python
for chunk in chain.stream({"text": "请用三句话介绍 LangChain"}):
    print(chunk, end="", flush=True)
```

## 总结

Runnable 的价值是统一组合方式：

```text
Prompt -> Model -> Parser
Retriever -> Prompt -> Model
Tool -> Agent
```

只要组件实现 Runnable，就可以继续组合成新的 Runnable。
