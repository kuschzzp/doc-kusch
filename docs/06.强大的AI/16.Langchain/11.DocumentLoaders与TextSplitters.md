# DocumentLoaders 与 TextSplitters

Document Loader 负责把外部数据加载成 `Document`。
Text Splitter 负责把长文档切成适合检索的小块。

## 加载文本文件

安装：

```bash
pip install -U langchain-community langchain-text-splitters
```

代码：

```python
from pathlib import Path
from langchain_community.document_loaders import TextLoader
from langchain_text_splitters import RecursiveCharacterTextSplitter

Path("demo.txt").write_text(
    "LangChain 是一个构建 LLM 应用的框架。\n" * 20,
    encoding="utf-8",
)

loader = TextLoader("demo.txt", encoding="utf-8")
docs = loader.load()

splitter = RecursiveCharacterTextSplitter(
    chunk_size=100,
    chunk_overlap=20,
)

chunks = splitter.split_documents(docs)

print(len(docs))
print(len(chunks))
print(chunks[0].page_content)
print(chunks[0].metadata)
```

## 加载目录

```python
from pathlib import Path
from langchain_community.document_loaders import DirectoryLoader, TextLoader

Path("demo_docs").mkdir(exist_ok=True)
Path("demo_docs/a.md").write_text("# A\nLangChain 文档 A", encoding="utf-8")
Path("demo_docs/b.md").write_text("# B\nLangChain 文档 B", encoding="utf-8")

loader = DirectoryLoader(
    "demo_docs",
    glob="**/*.md",
    loader_cls=TextLoader,
    loader_kwargs={"encoding": "utf-8"},
)

docs = loader.load()
print(len(docs))
print(docs[0].metadata)
```

## chunk_size 怎么选？

| 文档类型 | chunk_size | chunk_overlap |
| --- | --- | --- |
| FAQ | 300-800 | 50-100 |
| 普通文章 | 800-1200 | 100-200 |
| 技术文档 | 1000-1500 | 150-250 |
| 合同/法律 | 1000-2000 | 200-300 |

## 总结

RAG 前处理通常是：

```text
Loader -> Document -> TextSplitter -> Chunks
```
