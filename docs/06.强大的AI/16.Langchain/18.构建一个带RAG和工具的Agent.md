# 构建一个带 RAG 和工具的 Agent

这篇把前面的内容串起来：

- ChatModel
- Tool
- Retriever
- Agent
- RAG

## 安装

```bash
pip install -U langchain langchain-openai
export OPENAI_API_KEY="你的 key"
```

## 完整代码

```python
from datetime import datetime
from langchain.agents import create_agent
from langchain_core.documents import Document
from langchain_core.tools import tool
from langchain_core.vectorstores import InMemoryVectorStore
from langchain_openai import OpenAIEmbeddings

# 1. 准备知识库
docs = [
    Document(page_content="LangChain 用于构建 LLM 应用，包含模型、Prompt、Tools、RAG 和 Agent。"),
    Document(page_content="LangGraph 适合构建有状态、可分支、可循环的复杂 Agent 工作流。"),
    Document(page_content="LangSmith 用于调试、追踪和评估 LLM 应用。"),
]

embeddings = OpenAIEmbeddings(model="text-embedding-3-small")
vectorstore = InMemoryVectorStore.from_documents(docs, embeddings)
retriever = vectorstore.as_retriever(search_kwargs={"k": 2})

# 2. 定义 RAG 工具
@tool
def search_docs(query: str) -> str:
    """搜索 LangChain 相关知识库。

    当用户询问 LangChain、LangGraph、LangSmith、RAG、Agent 时使用。

    Args:
        query: 检索问题或关键词。
    """
    results = retriever.invoke(query)
    return "\n\n".join(doc.page_content for doc in results)

# 3. 定义普通工具
@tool
def get_current_date() -> str:
    """获取当前日期。"""
    return datetime.now().strftime("%Y-%m-%d")

# 4. 创建 Agent
agent = create_agent(
    model="openai:gpt-4o-mini",
    tools=[search_docs, get_current_date],
    system_prompt=(
        "你是一个中文 LangChain 技术助手。"
        "回答 LangChain 相关问题前，可以先搜索知识库。"
        "如果用户问日期，调用日期工具。"
    ),
)

# 5. 调用 Agent
result = agent.invoke({
    "messages": [
        {"role": "user", "content": "今天是哪天？顺便告诉我 LangSmith 是做什么的。"}
    ]
})

print(result["messages"][-1].content)
```

## 这个例子包含什么？

```text
用户问题
  -> Agent 判断需要哪些工具
  -> 调用日期工具
  -> 调用 RAG 检索工具
  -> 汇总结果并回答
```

## 可以继续扩展

- 把 `InMemoryVectorStore` 换成 Chroma。
- 接入 LangSmith 观察工具调用过程。
- 加入 `checkpointer` 支持多轮记忆。
- 用 LangGraph 控制复杂流程。
- 用 LangServe 暴露为 HTTP API。

## 总结

一个现代 LangChain 应用通常不是单个组件，而是组合：

```text
Model + Prompt + Tools + Retriever + Agent + Observability
```
