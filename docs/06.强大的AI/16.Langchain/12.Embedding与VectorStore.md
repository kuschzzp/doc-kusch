# Embedding 与 VectorStore

Embedding 把文本转成向量，VectorStore 保存向量并支持相似度搜索。

## InMemoryVectorStore

适合学习和 demo。

```python
from langchain_core.documents import Document
from langchain_core.vectorstores import InMemoryVectorStore
from langchain_openai import OpenAIEmbeddings

embeddings = OpenAIEmbeddings(model="text-embedding-3-small")

docs = [
    Document(page_content="LangChain 支持构建 RAG 应用。", metadata={"source": "a"}),
    Document(page_content="VectorStore 可以存储文本向量。", metadata={"source": "b"}),
]

vectorstore = InMemoryVectorStore.from_documents(docs, embeddings)

results = vectorstore.similarity_search("什么是向量库？", k=1)
print(results[0].page_content)
print(results[0].metadata)
```

## Chroma 持久化

安装：

```bash
pip install -U langchain-chroma
```

代码：

```python
from langchain_chroma import Chroma
from langchain_core.documents import Document
from langchain_openai import OpenAIEmbeddings

embeddings = OpenAIEmbeddings(model="text-embedding-3-small")

docs = [
    Document(page_content="Chroma 是一个常用的本地向量数据库。"),
    Document(page_content="LangChain 可以和 Chroma 集成。"),
]

vectorstore = Chroma.from_documents(
    documents=docs,
    embedding=embeddings,
    persist_directory="./chroma_demo",
    collection_name="demo",
)

results = vectorstore.similarity_search("Chroma 是什么？", k=1)
print(results[0].page_content)
```

## 注意事项

- 写入和查询要使用同一个 embedding 模型。
- 不同 embedding 维度不能混用。
- demo 可以用内存库，生产要用持久化向量库。
- 元数据要尽早设计好，后面检索过滤会用到。

## 总结

```text
文本 -> Embedding -> 向量 -> VectorStore -> similarity_search
```
