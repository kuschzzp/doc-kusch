## ReentrantLock 关键字

ReentrantLock 实现了 Lock 接口，和 synchronized 关键字类似，与synchronized都是可重入锁。但是 ReentrantLock 比 synchronized
更加灵活，提供了一些 synchronized 关键字不具备的特性，例如指定锁是否为公平锁，true为公平锁。

### 1. 比较 synchronized 增加的特性

- lock.lockInterruptibly()：可以中断当前等待的线程，转而处理其他事情。

- Condition 接口 精准控制线程的执行。例子如下：

```java
import java.util.concurrent.locks.Condition;
import java.util.concurrent.locks.Lock;
import java.util.concurrent.locks.ReentrantLock;

public class TestConditionLock {
    public static void main(String[] args) {
        TestCondition testCondition = new TestCondition();
        new Thread(() -> {
            for (int i = 0; i < 10; i++) {
                testCondition.testA();
            }
        }, "A").start();
        new Thread(() -> {
            for (int i = 0; i < 10; i++) {
                testCondition.testB();
            }
        }, "B").start();
        new Thread(() -> {
            for (int i = 0; i < 10; i++) {
                testCondition.testC();
            }
        }, "C").start();
    }
}

class TestCondition {
    private Lock lock = new ReentrantLock();
    private Condition conditionA = lock.newCondition();
    private Condition conditionB = lock.newCondition();
    private Condition conditionC = lock.newCondition();

    //1A 2B 3C
    private int number = 1;

    public void testA() {
        lock.lock();
        try {
            while (number != 1) {
                conditionA.await();
            }
            System.out.println(Thread.currentThread().getName() + "我直接走AAA");
            number = 2;
            conditionB.signal();
        } catch (Exception e) {
            e.printStackTrace();
        } finally {
            lock.unlock();
        }
    }

    public void testB() {
        lock.lock();
        try {
            while (number != 2) {
                conditionB.await();
            }
            System.out.println(Thread.currentThread().getName() + "我直接走BBB");
            number = 3;
            conditionC.signal();
        } catch (Exception e) {
            e.printStackTrace();
        } finally {
            lock.unlock();
        }
    }

    public void testC() {
        lock.lock();
        try {
            while (number != 3) {
                conditionC.await();
            }
            System.out.println(Thread.currentThread().getName() + "我直接走CCC");
            number = 1;
            conditionA.signal();
        } catch (Exception e) {
            e.printStackTrace();
        } finally {
            lock.unlock();
        }
    }
}
```
